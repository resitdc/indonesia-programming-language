Rakoda Programming Language

Bahasa Pemrograman Pertama Berbasis Bahasa Indonesia untuk Siswa SMK.

Mulai Belajar GitHub

Instalasi

RPL adalah bahasa pemrograman yang berjalan secara native (menggunakan Rust VM) dengan performa tinggi. Anda dapat menginstalnya di berbagai sistem operasi.

Windows

Jalankan file installer rpl-setup.exe dan ikuti petunjuk di layar.

macOS (Apple Silicon / Intel)

curl -sSL https://rpl-lang.org/install.sh | bash

Memulai (Hello World)

File RPL memiliki ekstensi .rpl atau .ipl. Mari buat file pertama Anda bernama halo.rpl.


// Ini adalah komentar
cetak "Halo Dunia!"

Jalankan kode tersebut dengan terminal:

rpl run halo.rpl

Variabel & Tipe Data

Gunakan kata kunci buat untuk mendeklarasikan variabel.


buat nama = "Budi"        // Teks (String)
buat umur = 17            // Angka (Number)
buat lulus = benar        // Boolean (benar / salah)
buat nilai = [90, 85, 88] // Array
buat siswa = {            // Kamus (Object/Dictionary)
    "nama": "Andi",
    "kelas": 12
}
buat kosong_val = kosong  // Tipe Kosong (Null)

Kondisional (Jika-Maka)


buat nilai = 85

jika nilai > 90 maka {
    cetak "Grade A"
} jika tidak nilai > 80 maka {
    cetak "Grade B"
} jika tidak {
    cetak "Grade C"
} selesai

Perulangan

Perulangan Selama (While)


buat i = 1
selama i <= 5 {
    cetak i
    buat i = i + 1
} selesai

Perulangan Ulangi (For)


buat angka = [10, 20, 30]
ulangi x dalam angka {
    cetak x
} selesai

Fungsi

Mendeklarasikan fungsi dan mengembalikan nilai menggunakan kata kunci fungsi dan kembalikan.


fungsi tambah(a, b) {
    kembalikan a + b
} selesai

buat hasil = tambah(5, 10)
cetak hasil // Output: 15

Error Handling

Menangani kemungkinan error atau panic saat program berjalan.


coba {
    buat hasil = 10 / 0
    cetak hasil
} tangkap err {
    cetak "Terjadi kesalahan: " + err
} selesai

// Melempar error buatan sendiri
lempar "File tidak ditemukan!"

Pengembangan Web: Routing HTTP

RPL dilengkapi dengan framework web native yang sangat cepat. Anda dapat membuat server HTTP dengan mudah.


impor "web"

web.get("/", fungsi(req) {
    kembalikan "
Halo dari RPL Web Server!
"
} selesai)

web.post("/api/login", fungsi(req) {
    // req.tubuh berisi payload POST
    // req.params berisi parameter URL dinamis
    buat res = {
        "status": "sukses",
        "pesan": "Login berhasil"
    }
    kembalikan res // Otomatis dikembalikan sebagai JSON
} selesai)

// Menjalankan server di port 8080
web.jalankan(8080)
Info: HTTP Method Lengkap

RPL Web mendukung semua HTTP method utama: web.get, web.post, web.put, web.patch, dan web.delete.

Menyajikan File Statis

Untuk menyajikan gambar, CSS, HTML, atau JavaScript, gunakan web.statis.


impor "web"

// Akses /aset/style.css akan memanggil ./public/style.css
web.statis("/aset", "./public")

web.jalankan(3000)

RPL CLI Reference

Perintah Deskripsi
rpl -v Melihat versi bahasa RPL
rpl run <file> Menjalankan kode RPL
rpl serve <file> Menjalankan file rpl.html sebagai server template dinamis
rpl fmt <file/folder> Meratakan dan memformat otomatis struktur kode RPL

Manajer Paket (RPL Modules)

RPL memiliki sistem manajemen paket sendiri berbasis repository GitHub.


# Membuat proyek baru dan file rpl.json
rpl init

# Menginstal modul dari GitHub ke folder rpl_modules/
rpl instal github:user/repo

# Menghapus modul
rpl hapus user/repo

File rpl.json akan terlihat seperti ini:


{
  "nama": "proyek-keren",
  "versi": "1.0.0",
  "titik_masuk": "main.rpl",
  "dependensi": {
    "repo": "github:user/repo"
  }
}
